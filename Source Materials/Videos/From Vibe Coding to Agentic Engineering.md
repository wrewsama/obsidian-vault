- The Software 3.0 Mindset shift
    - Interpreters -> LLMs
    - Code -> Context
    - LLMs perform increasingly more _information processing_ (much broader than just coding)
- **Verifiability** determines how well you can automate something with an AI agent
- **Agentic Engineering vs Vibe Coding**: Increasing development efficiency WITHOUT comprising the quality bar
- tips
    - work with the agent to create a full spec before letting it work (not just using Plan mode)
    - set up agent-first infrastructure (e.g. tools usable by agents, docs that can be easily accessed by agents)
- What agents can't do (yet?): taste, design, judgement, _engineering_

> We're not building animals, we're summoning ghosts
- Agents are statistical, not intrinsically motivated

> You can outsource your thinking, but you can't outsource your understanding
