- Allows software written using the IPv6 API to work with IPv4 addresses [[IPv4 vs IPv6]]
- `aaa.bbb.ccc.ddd` => `::ffff:aaa.bbb.ccc.ddd`
- can also use the usual ipv6 hex notation `::ffff:aabb:ccdd` (each 8-bit decimal number is 2 hexadecimal digits)
