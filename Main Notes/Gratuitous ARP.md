## what
- Different from [[ARP]]
- An unsolicited (_gratuitous_) broadcast by a host stating its IP address and MAC address

## benefits
- detect IP conflicts
- update other machines' ARP table (useful when the IP changes NIC, which changes its MAC address)
- switch learning
