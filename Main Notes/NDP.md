- Neighbour Discovery Protocol
- For IPv6, replaces [[ARP]], some parts of [[DHCP]], and some ICMP mechanisms
- Capabilities
    - neighbour discovery (MAC addresses)
    - neighbour unreachability detection
    - router discovery
    - IPv6 address auto configuration
    - duplicate address detection
    - redirection
