- PEMDAS
- left-associative for almost all same-level operations
    - `a * b // c * d` = `((a * b) // c) * d`
- **EXPONENTIATION IS THE EXCEPTION**. It's right-associative
    - `a ** b ** c == a ** (b ** c)`
