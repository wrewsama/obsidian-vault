- address length 32 bit vs 128 bit
    - IPv4: 4 8-bit decimal numbers e.g. `192.168.1.1`
    - IPv6: 8 groups of 4 hexadecimal digits e.g. `2001:0db8:0000:0000:0000:ff00:0042:8329`
        - can use `::` to shorten contiguous `0`s, but you can only use it once in an address (e.g. `2001:0db8::ff00:0042:8329`)
        - using 2 would make it impossible to tell how many `0`s are in each one
- ipv6 solves the IP address exhaustion problem
